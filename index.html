Side Projects

Gimmicode

Gimmicode is a simple utility giving the Unicode code point of an inputed character.

The tiny project serves as a way to discover new language and framework, as it is often rewrote with the language I am trying to get acquainted with. The current version is written in Go and Redis.

Both the Go and the original command-line utility are open source and available on Github.

Abjapp

Abjapp is an iPhone application —iPad update coming soon—, teaching people the Arabic alphabet.

I designed the user interface using Sketch and wrote the code in Objective-C (no Swift for this one).

This application was fun to build, it allowed me to combine my interest in both iOS and writing systems.
