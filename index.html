Abjapp

June 2015 to Ongoing
Technologies
Version 1.0 Objective-C Core Data Version 1.2 Swift Realm
Download from the App Store

Abjapp is an iPhone application teaching the Arabic alphabet.

The Arabic alphabet contains 28 letters. Each lesson focuses on three specific letters and must be validated by a quiz to progress. Knowledge can also be practiced on-demand, with random quizzes.

Abjapp offers details about each letters: their names, how to pronounce them and how to write them depending on their positions within a word.

actiTENS is composed of two parts:

Sophisticate is a little JavaScript utility that I have often used to clean and add proper metadata to the output of vector editors. The generated SVGs tend to be messy and lack proper classes / ids to be usable, once inlined into an html document.
