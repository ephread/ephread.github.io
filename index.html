Sophisticate

Clean SVGs and use them in interactive web pages
Date
May 2017 to Ongoing
Technologies
Javascript
See on Github

Sophisticate is a little javascript utility that I have often used to clean and add proper metadata to the output of vector editors. The generated SVGs tend to be messy and lack proper classes / ids to be usable, once inlined into an html document.

Sophisticate supports different modification rules, and can also run SVGO to further clean SVGs. Given how often I work with SVG, this little utility is a huge time saver.
